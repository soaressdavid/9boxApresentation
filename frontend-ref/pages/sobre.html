Portal de Gestão de Pessoas

Sistema completo para avaliação de desempenho, gestão de competências e desenvolvimento de talentos

O que é o sistema?

O Portal de Gestão de Pessoas é uma plataforma desenvolvida para facilitar a avaliação de desempenho, gestão de competências e acompanhamento do desenvolvimento profissional de colaboradores e gestores.

Com foco em avaliações anônimas e bidirecionais, o sistema permite que gestores avaliem colaboradores e vice-versa, promovendo uma cultura de feedback contínuo e transparente.

Principais Funcionalidades

Gestão de Usuários

Cadastro e gerenciamento de colaboradores, gestores e administradores com controle de permissões

Avaliações Anônimas

Sistema de avaliações bidirecionais com anonimato garantido para feedback honesto

Nine Box Grid

Matriz de desempenho e potencial para identificação de talentos e planejamento sucessório

Competências

Gestão de competências técnicas, comportamentais e de liderança

Relatórios

Dashboard estratégico com indicadores de desempenho e análises detalhadas

Avaliação 180° e 360°

Avaliações multidirecionais para visão completa do desempenho

Tecnologias Utilizadas

O sistema foi desenvolvido com tecnologias modernas e robustas, garantindo performance, segurança e escalabilidade.

Node.js + Express
PostgreSQL + Prisma ORM
JavaScript ES6+
HTML5 + CSS3
JWT Authentication
Bcrypt Encryption
